## KiCad Design Files

This folder contains the KiCad design files for the analog synthesizer PCB.

### File descriptions

- **Synthesizer.kicad_sch**  
  KiCad schematic file containing the complete circuit design, including the VCO, exponential converter, summing amplifier, LFO, and supporting circuitry.

- **Synthesizer.kicad_pcb**  
  KiCad PCB layout file defining the board outline, component placement, routing, and copper layers.

- **Synthesizer.kicad_pro**  
  KiCad project file that links the schematic and PCB layout and stores project-specific settings.

- **Synthesizer_fab_file_11_01_26.zip**  
  Fabrication files exported from KiCad, including Gerber files and drill data, ready for PCB manufacturing.
